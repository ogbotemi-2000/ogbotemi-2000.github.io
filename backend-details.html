Idea Expansion

TL:DR; Users provide their API keys and get services that fetch and display market data by using templates or their own setups

This service is being expanded upon into a full-fledged SAAS that offers admin chart areas, dashboards, infographics and details plotted, generated and surmised from the fields configured prior by users in the API response of crypto market platforms like CoinGeckco

The target is to sell easily created and customizable portfolios of crypto or financial market analysis data to techy and non-techy users.

The idea will evolve into one that returns dashboards as HTML web pages for JSON data sent in an API request to it. It will also offer building blocks of charts and infographics that shortens development time and ensures flexibility

It will be a major improvement on how HubSpot uses JSON data to render parts of a webpage UI - it will include feature flags, customizable composable UI, fast builds, REST API, design spaces, plug-and-play features without vendor lock in
